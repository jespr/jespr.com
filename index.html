Personal projects

The following is a list of existing/past personal projects I have been working on.

  • elixir, postgres, sendgrid, tachyons Formbackend Formbackend makes it so people can add a form to their static HTML page - and Formbackend will capture responses being submitted. Good when using Jekyll, Github pages etc.
  • elixir, nginx, postgres, tachyons Tabs & Spaces Tabs & Spaces is a blogging platform focused on developers. I mostly write technical content and I found a lot of the existing blogging solutions lacking. I wanted to create something where content would be surfaced between different blogs - but things like syntax highlighting and grouping posts in to series would be possible.
  • rails, postgres, react-native Spendiings An app to track spending. Didn't like the existing solutions out there like Mint etc. Decided to focus on income periods and the amount of money you had within each period (if you get paid bi-weekly, and you got paid $1,000 - then you would have $1,000 to spend in the next two weeks). It would automatically deduct recurring bills, from the periods based on the frequency. Wrote an integration to pull from your bank account automatically using Plaid.
  • rails, postgres MyDailyLife Small app that would send you an email at a set daily time to remind you to write your daily journaling entry.
  • rails, postgres, react-native Photoshare A photo sharing application, users create "moments" and all invited users could upload photo to a given moment. Had a small mobile application as well, from where users could upload photos and view moments.
